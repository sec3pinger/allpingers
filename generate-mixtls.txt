The files allpingers.txt and mix.tls are generated by a handful of Perl scripts that were created many years ago by Richard Christman, the creator of Quicksilver Lite remailer client. Richard shared the scripts with me and I have been using them, on my VPS server, to generate and publish an up-to-date mix.tls file.

If you would like to run these scripts yourself and generate your own mix.tls file visit the URL below to download all the necessary Perl scripts. There are set up instruction in the README file.
