The files allpingers.txt and mix.tls are generated by a handful of Perl
scripts that were created many years ago by Richard Christman, the creator
of Quicksilver Lite remailer client.

Richard shared the scripts with me and I have been using them for years,
on my VPS server, to generate and publish the up-to-date mix.tls file.

If you would like to run these scripts yourself and generate your own mix.tls
file visit the URL below to download howto-generate-mixtls.zip. There are
setup instructions in the /mixtls/READ.ME file.

https://app.box.com/s/d3maei3484ruhiez3frq63sf3rodytfd
